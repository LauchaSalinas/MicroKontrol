# MicroKontrol
Arduino project for adding sustain and expression pedal to the MicroKorg XL

Used Libraries:
SevenSegmentTM1637
MIDI_Library
EPPROM

Required materials:
1x Arduino Nano
1x 18650 baterry
1x TP4056 Charger module
1x 4 digits 7-segments display
2x 10k Potenciometers
1x Rotary encoder KY-040
1x TS Jack
1x TRS Jack
